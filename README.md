# WFLAer.space Website

A modern, responsive website for WFLAer.space, built with React, TypeScript, and Tailwind CSS. This website serves as the main platform for our non-profit organization dedicated to enhancing education at WFLA International through technology.

## Features

- 🚀 Modern, responsive design
- 🎨 Beautiful animations and transitions
- 📱 Mobile-first approach
- 🔍 SEO-friendly
- ⚡ Fast loading times
- 🌐 Multi-page application with React Router

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router
- Lucide Icons
